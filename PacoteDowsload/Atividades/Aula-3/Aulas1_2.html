  2. Descreva com suas palavras e entendimento o que é URL. Diferencie e exemplifique
    URL absoluta e URL relativa. URL é o localizador uniforme de recursos na internet,

             O tipo absoluto de URL é aquele que contém protocole, servidor, domínio e o

  3. Antes da leitura da Aula 01, você sabia diferenciar os termos: home-page, página web
    e web site? Justifique sua resposta e ainda comente sobre cada um deles de forma a
    diferenciá-los.

             Home-page é a página inicial de um site, nela estão os títulos e ícones que
    levarão a todos os assuntos e arquivos do web site. Página web, por sua vez, é um
    documento de hipertexto com todos os recursos de HTML, CSS e JavaScript. Juntos,
    home-page e página web ajudam a formar um web site, esse precisa ser colocado em
    um servidor e estar disponível na internet caso contrário será apenas um aglomerado
    documentos hipertextos.

  4. Dentro do contexto web, pesquise, defina e descreva com suas palavras e
    entendimento sobre os termos: Front end e Back end.

             Front end é a parte visual da interface como imagens, textos, espaçamentos,
    ícones animados que abrem mais títulos ao passarmos o cursor sobre ele. O HTML é
    toda parte estrutural da página que precisa das CSS para ficar devidamente ordenada,
    somando a eles o JavaScript que dá dinamismo aos títulos e demais objetos da página
    web.

             Nos bastidores o back end faz todo trabalha para que o usuário tenha os
    resultados esperados. Um formulário aos ser preenchido passa pelo crivo do back end
    para ser validado, nessa fase serão avaliados se os campos foram preenchidos de forma
    correta, antes de buscar qualquer informação no banco de dados, caso contrário
    aparecerá uma mensagem de erro para o usuário.

  1. Explique como ocorre o processo de relacionamento cliente/servidor web.
